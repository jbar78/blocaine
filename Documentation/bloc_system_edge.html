1Interface graphique

 

2Fonction

Le bloc <edge> détecte les fronts montants et/ou descendant en comparant l’entrée « in » avec la sortie « in(n-1) ». Noter qu’à la première exécution la sortie « out » sera « invalide » puis que la sortie « in(n-1) » n’est pas encore disponible. Les entrées « ⚙Rise » et « ⚙Fall » permettent de sélectionner les types de front à détecter.

3Les entrées

  • •« in » : C’est la variable de type bool dont on cherche à détecter les changements d’état. La valeur par défaut est 0 (type int) 

  • •« ⚙Rise » : Si True les fronts montants seront détectés. La valeur par défaut est True (type bool), 

  • •« ⚙Fall » : Si True les fronts descendants seront détectés. La valeur par défaut est False (type bool) 

4Les sorties

  • •« out » : Est à True si un front est détecté 

  • •« in(n-1) » : Représente l’état de l’entrée « in » de l’exécution précédente, c’est une variable mémorisée, ce qui explique la couleur de fond vert pomme. 

5Types

Toutes les entrée/sorties sont de type bool.
