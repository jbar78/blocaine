1Interface graphique

 

2Fonction

Le bloc <dt> calcule :

  • •Le temps de cycle en seconde mesuré entre deux exécutions, le calcul est basé sur la différence entre le nombre de tic lors de l’exécution par rapport au nombre de tic de l’exécution précédente qui est mémorisée dans la sortie « t(n-1) ». 

3Les entrée(s)

Il n’y a aucune entrée.

4Les sorties

  • •« dt » : temps de cycle entre deux exécutions, Notez qu’à la première exécution cette sortie est invalide. 

  • •« t(n-1) » : nombre de tic lors de la dernière exécution, Notez que cette sortie est une variable mémorisée de cycle en cycle, ce qui explique la couleur de fond vert pomme. 

5Types

La sortie « dt » est de type : float.

La sortie « t(n-1) » est de type : int.

6Usage

Voir aussi le bloc <dt_task> qui est moins gourmand en ressources.

Comme la sortie « t(n-1) » correspond au nombre de tic lors de l’exécution, vous pouvez mesurer le temps d’exécution, en utilisant plusieurs blocs <dt>.
