2Fonction

Le bloc « memory » est une mémoire Set-Reset (ou bascule RS).

Le paramètre « ⚙reset » permet de sélectionner la priorité entre l’entrée « set » et « reset » dans le cas ou ces deux entrées soient à True en même temps.

3Les entrées

  • •« set » : La valeur par défaut est False (type bool) 

  • •« reset » : La valeur par défaut est False (type bool) 

  • •«  ⚙reset » : c’est un paramètre, La valeur par défaut est True (type bool) 

    • ◦ si True :  l’entrée « reset » est prioritaire 

    • ◦si False : l’entrée « set » est prioritaire 

4La sortie

  • •« out » : est l’état de mémoire, c’est une variable mémorisée qui peut être initialiser, ce qui explique la couleur de fond vert pomme 

5Types

les entrées et la sortie sont toutes de type : bool
