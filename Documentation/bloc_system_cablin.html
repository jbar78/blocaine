2Fonction

Le bloc <cablin> permet de fusionner deux variables en une. Par analogie cela revient à former un câble contenant deux conducteurs.

Concrètement la variable de sortie « cable » est un tule contenant deux éléments l’entrée « cable » et l’entrée « wire ».

Se reporter à la documentation Python pour plus de détails concernant les tules.

3Les entrées

  • •« cable » : La valeur par défaut est 0 (type int) 

  • •« wire » : La valeur par défaut est 0 (type int) 

4La sortie

  • •« cable » : c’est le tule contenant les deux entrées 

5Types

Le type des entrées peut être quelconque, le sortie est un tule.

6Usage

Cela permet de créer des groupes de variables, chaque groupe pouvant être utilisé comme une variable simple.  

Pour dégroupe, utilisez le bloc <cableout>.
