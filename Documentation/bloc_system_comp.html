2Fonction

Le bloc <comp> est l’équivalent des opérateurs >, == et < en Python, il permet soit :

  • •La comparaison de 2 nombres 

  • •La comparaison de 2 éléments (chaînes de caractères, list, tuple...)  

Se reporter à la documentation Python pour plus de détails.

3Les entrées

  • •« a » : La valeur par défaut est 0 (type int) 

  • •« b » : La valeur par défaut est 0 (type int) 

4Les sorties

  • •« a  > b » : est à vrai si « a » est supérieur à « b », sinon est à faux. 

  • •« a  == b » : est à vrai si « a » est égal à « b », sinon est à faux. 

  • •« a  < b » : est à vrai si « a » est inférieur à « b », sinon est à faux. 

5Types

les entrées doivent être de même nature.
