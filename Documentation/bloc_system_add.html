2Fonction

Le bloc <add> est l’équivalent de l’opérateur « + » en Python, il permet soit :

  • •l’addition de 2 nombres 

  • •La concaténation de 2 éléments (chaînes de caractères, list, tuple)  

Se reporter à la documentation Python pour plus de détails.

4La sortie

  • •« out » : Son type est fonction du type des entrées 

5Types

Si  une des entrée est de type : int, float ou complex, l’autre entrée doit aussi faire partie de l’un de ces 3 types.

Si une des entrées est de type « chaîne de caractères », l’autre entrée doit être de même type

Si une des entrées est de type « list », l’autre entrée doit être de même type

Si une des entrées est de type « tuple », l’autre entrée doit être de même type
