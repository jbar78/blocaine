2Fonction

Le bloc « select » permet de sélectionner une entrée parmi deux à l’aide de l’entrée « gate ».

  • •si « gate » est à faux : la sortie « out » est l’image de l’entrée « 0 » 

  • •si « gate » est vrai : la sortie « out » est l’image de l’entrée « 1 » 

Noter que l’Exécuteur n’évalue pas l’entrée non sélectionnée, celle-ci n’étant pas utilisée. On peut considérer ce comportement comme l’équivalent d’un saut conditionnel dans un langage structuré.

3Les entrées

  • •« 0 » : La valeur par défaut est 0 (type int) 

  • •« 1 » : La valeur par défaut est 0 (type int) 

4La sortie

  • •« out » : Son type est fonction du type des entrées 

5Types

Les entrées peuvent être de n’importe quel type du moment que c’est un type accepté par le langage Python.

La sortie est du même type que l’entrée sélectionnée.
